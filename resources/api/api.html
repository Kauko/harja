Harja API API documentation version 0.0.5

https://harja/api/0.0.5

Kuvaus

Harja -järjestelmän julkinen API

API on jaettu neljään pääosioon:

  1. urakat:
    • Urakoiden perustietojen haku joko urakoitsijan id:llä tai yksittäisen urakan id:llä
    • Urakan seurantatietojen kirjaus: toteumat, poikkeukset, tarkastukset ja havainnot
  2. varusteet:
    • Varustetietojen haku Tierekisteristä
    • Varustetietojen päivitys Tierekisteriin.
  3. seuranta:
    • Reaaliaikaisten seurantatietojen, kuten esim. aura-auton liikkeiden kirjaus
  4. ilmoitukset:
    • Ilmoitusten haku urakoitsijakohtaisesti
    • Ilmoitusten ilmoitusten kuittaaminen

Tekninen kuvaus

Harja -järjestelmän rajapinta on toteutettu REST-API:na, jossa tieto liikkuu JSON-formaatissa.

Tietoliikenne Harjan palvelimille kulkee Liikenneviraston keskitetyn gatewayn kautta (Oracle Enterprice Gateway). Siirtoprotokollana käytetään HTTPS. Käyttäjän tunnistaminen tehdään käyttäen HTTP Basic autentikaatiota tunnus-salasana -parina. Liikennevirasto hallinnoi järjestelmä-/urakoitsijakohtaisia tunnuksia.

Sijaintitiedot

Kaikki koordinaattisijainnit ilmoitetaan Harjaan EUREF-FIN koordinaatistossa. Koordinaattipisteiden lisäksi vastaanotetaan suunta joka on aseteluku väliltä: 0-360.
